# Shoks SAT Landing Page

This is a [Next.js](https://nextjs.org) project for Shoks SAT tutoring services, featuring a modern landing page with comprehensive refactoring and design improvements.

## Recent Updates

- ✅ Removed dark theme completely
- ✅ Fixed cursor styles for all interactive elements
- ✅ Hidden scrollbar globally
- ✅ Swapped Results/Reviews order in navigation
- ✅ Standardized button radius to rounded-sm
- ✅ Unified all transitions to 300ms ease-out
- ✅ Improved mobile experience with centered footer and hamburger menu animations
- ✅ Created reusable components (ButtonNav, Button)
- ✅ Consolidated navbar components
- ✅ Fixed TypeScript errors
- ✅ Added not-found page

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Project Structure

- `/src/components/` - Reusable React components
- `/src/app/` - Next.js app directory with pages and layouts
- `/public/` - Static assets including logos and images
